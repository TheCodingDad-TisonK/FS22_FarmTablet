# Farm Tablet — FS22 Mod

**Farm Tablet** is a script mod for *Farming Simulator 22* that adds a **compact in-game tablet interface** showing useful information and tools for your farm. It’s a work-in-progress project that aims to grow into an all-in-one farm management tablet with apps and real data displays.  
Original mod page: https://www.kingmods.net/en/fs22/mods/73757/farm-tablet

---

## 📌 Overview

This mod introduces a tablet UI element into FS22 that you can toggle in-game to view and interact with various stats and functions. It’s currently in **active development**, meaning features are being refined, visual icons are being added, and more apps are planned.

---

## 🧠 Features

- 📱 **In-game Tablet UI** — A tablet display that opens with a key bind (`T` by default).
- 🗺️ **Dashboard and Apps** — Contains apps like weather and other utilities (still expanding).
- 🚜 **Vehicle Detection** — Shows data related to your vehicles.
- 🔌 **Integrates with Other Mods** — Works with **Income Mod** and **Tax Mod** for enhanced data views.
- 🛠️ **Planned Improvements** — UI refinement, icons, sound effects, more apps, and better interaction.
